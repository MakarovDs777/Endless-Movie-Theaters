Бесконечная видеотека бэкрумса, это неограниченный ютуб с бесконечным количеством видео каждое из которых может быть которое только снято, и соответственно продолжительность каждого видео, и самих видео в ней от одного до бесконечности включая саму бесконечность, бесконечная закулисная видеотека разделена на стеллажи, который обозначаются числом, равным количеству продолжительности видео, например, если это стеллаж № 1, то все видео в нем с одной секундой, и все варианты размещения всех возможных кадров всех всевозможных расположений пикселей в этом кадре каждую секунду, если это стеллаж № 2 тогда в нём все видео по 2 секунды и все варианты расположения всех кадров в этом видео которые только могут быть и т.д.
